# Opis

Badany wykazuje wysoką tolerancję wobec ludzi przy jednocześnie niskiej tolerancji wobec błędów poznawczych, niespójności logicznych oraz słabej argumentacji.

Ocena osoby i ocena jej rozumowania wydają się funkcjonować jako dwa odrębne procesy.

W rezultacie badany może jednocześnie:

- szanować rozmówcę,
- zachowywać życzliwość,
- wykazywać cierpliwość,

oraz równocześnie:

- kwestionować argumenty,
- wskazywać błędy logiczne,
- podważać założenia,
- krytykować modele decyzyjne.

Krytyka kierowana jest przede wszystkim wobec treści, procesu lub sposobu rozumowania, a nie wobec człowieka.

Predyktor rozróżnia tolerancję wobec ludzi od niskiej tolerancji wobec błędów poznawczych.

## Analiza Jakościowa

- Badany deklaruje wysoką tolerancję wobec ludzi.
- Najbardziej irytującym czynnikiem jest nielogiczność.
- Badany regularnie oddziela ocenę osoby od oceny argumentu.
- W rozmowach częściej pojawia się analiza jakości rozumowania niż ocena intencji rozmówcy.
- Krytyka najczęściej dotyczy procesu decyzyjnego lub modelu rzeczywistości.

# Notatki

Predyktor nie opisuje konfliktowości.

Predyktor nie opisuje również potrzeby krytykowania innych ludzi.

Opisuje sposób rozdzielania:

- osoby,
- argumentu,
- modelu,
- decyzji.

Najbardziej prawdopodobny wzorzec działania można opisać jako:

1. Oceń człowieka.
2. Oceń argument.
3. Oceń model.
4. Nie zakładaj automatycznie, że błędny model oznacza błędnego człowieka.

Predyktor może wyjaśniać:

- pozorne rozbieżności pomiędzy Big Five i HEXACO,
- wysoką gotowość do debat merytorycznych,
- preferencję dla rozmów opartych na argumentach,
- koncentrację na jakości procesu decyzyjnego.

Na obecnym etapie jest to jeden z najlepiej wspieranych predyktorów interpersonalnych w całym modelu.

CAL-001 doprecyzowuje rozróżnienie między tolerancją wobec ludzi a niską tolerancją wobec błędów poznawczych.
